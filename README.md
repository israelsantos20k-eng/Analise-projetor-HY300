# Análise de Qualidade – Projetor HY300

Este repositório contém um relatório de análise qualitativa desenvolvido durante aula da EBAC.

## 📌 Objetivo
Avaliar a qualidade de um produto com base em critérios como material, durabilidade, usabilidade e design.

## 📚 Conteúdos aplicados
- Garantia da Qualidade (QA)
- Controle de Qualidade (QC)
- Experiência do Usuário (UX)
- Conceitos de qualidade de software
